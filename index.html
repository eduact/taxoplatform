Conexión a Google Sheets
1
Crea un Google Sheet
Ve a sheets.google.com → crea una hoja nueva → copia el ID de la URL:
docs.google.com/spreadsheets/d/ESTE_ID/edit
2
Obtén un API Key de Google Cloud
console.cloud.google.com → nuevo proyecto → Habilitar "Google Sheets API" → Credenciales → Crear → Clave de API. Restringe la clave a Google Sheets API. Haz el Sheet público de lectura (Compartir → "Cualquier persona con el enlace" puede ver).
3
Crea un Apps Script Web App (para escritura)
En el Sheet → Extensiones → Apps Script → pega el código del README → Implementar → App web → Ejecutar como: Yo · Acceso: Cualquiera → copia la URL.

? ¿Cómo preparar tu archivo?
1
Descarga el template
Usa el botón de abajo para obtener el archivo Excel con el formato correcto para el cliente activo.
2
Llena el template
El archivo tiene una pestaña por nivel (Campaign, Placement, Creative). En cada pestaña llena dos columnas:
Platform Plataforma de la taxonomía (META, GOOGLE, TIKTOK…)
Taxonomy_Name El nombre completo de la taxonomía a evaluar
3
Sube y valida
Arrastra o selecciona el archivo completado. El sistema detecta automáticamente la pestaña correspondiente a cada nivel.
